The TOC is a library for the project [document2](https://github.com/cable8mm/document2).

## Features

- [x] Laravel toc
- [x] Samsung Tizen toc
- [x] Naver clova ai toc
- [x] Rhymix toc

## Installation

```shell
composer require cable8mm/toc
```

## Testing
